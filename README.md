# **git命令**  
git init  
git add README.md  
git commit -m "first commit"  
git remote add origin git@github.com:zhouming735/python3-blog.git  
git push -u origin master  

# **环境准备**
+ python3.6 安装  
+ pyCharm IDE 安装  
+ web项目结构      
├──backup //备份目录  
├──conf   //配置目录  
├──dist   //发布目录  
├──doc    //文档说明目录  
├──www    //站点文件目录  
│ ├──static  //静态文件目录  
│ ├──templates  //模板文件目录  
├──gitignore   //文件忽略规则设置文件   
├──LICENSE   //许可文件  
├──README  //项目描述文件
